=== КЕЛЕСІ ҚАДАМДАР ===

1. Дерекқорды жаңарту:
   cd /home/ubuntu/s7-platform
   npx prisma migrate dev --name add_news_module

2. Сервер мен клиентті іске қосу:
   Terminal 1: cd server && npm run dev
   Terminal 2: cd /home/ubuntu/s7-platform && pnpm dev

3. Браузерде тестілеу:
   - Админ: http://localhost:3000/admin/news
   - Public: http://localhost:3000/news
